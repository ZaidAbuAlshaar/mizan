# ميزان MIZAN — دليل الهوية و UI/UX

> مرجع Max 2 (Frontend/Design). يكمّل الدستور §١٣ (الشاشات الخمس + الهوية). الهدف: **"غرفة تحكّم فضائية"** تُقنع حَكَماً في ثوانٍ — وضوح وتباين وحركة هادفة، لا زخرفة.

---

## 1) الهوية البصرية (المرجع الملزم)
- **المزاج:** غرفة تحكّم/مهمّة فضائية — جدّية، دقيقة، "بيانات حيّة".
- **الألوان:** خلفية كحلي/فحمي عميق · لمسات **تركوازية** · أرقام كبيرة عالية التباين · حالات P4 الموحّدة **🔴 ≥٧٠ · 🟠 ٤٠–٦٩ · 🟢 <٤٠**. اجعلوها **Tailwind tokens** (من TM-11).
- **الخطوط:** **Almarai** (عناوين/أرقام) + **Tajawal** (نصوص). أوزان ثقيلة للأرقام البطلة.
- **الخريطة:** **MapLibre GL** بنمط داكن (dark basemap) — لا مفاتيح/فواتير.
- **اللغة:** **عربي-RTL أولاً** مع مبدّل EN؛ كل visualization ثنائي اللغة (الدستور §٢٤).
- **الشعار:** كفّتا ميزان بقوس مدار + قمرا GRACE فوق خط منسوب ماء (SVG).

## 2) الشاشات الخمس — ملاحظات UI سريعة (الدستور §١٣)
1. **الخريطة الوطنية:** أحواض ملوّنة بالصحّة + نقاط 🔴🟠🟢 + شريط جانبي يمين (أعلام، م³، آخر تحديث).
2. **تفاصيل الحوض:** منحنى GRACE هابط + خط تنبّؤ متقطّع + **نقطة حمراء وامضة** (العتبة الحرجة) + آلة الزمن ٢٠١٦↔٢٠٢٦.
3. **طابور التفتيش (قلب المنتج):** جدول مرتّب بالدرجة → ضغطة صفّ تفتح **بانل الدليل** (NDVI + قبل/بعد + مكوّنات الدرجة + زر الحالة).
4. **عدّاد الأثر:** ٣ أرقام ضخمة تتصاعد (م³ · أشخاص · دنانير) + شريط مقارنة الناقل الوطني.
5. **المنهجية:** المصادر + model card + الحدود — ثنائية اللغة.

- **Nucleo Patterns** — خلفيات SVG (شبكات/نقاط) لطبقة "غرفة التحكّم" خلف البطاقات.
- **Skiper UI · Cult UI · Pixlo · Ali Imam** — مكوّنات حركة/بطاقات/hero جاهزة فوق Tailwind؛ خذوا منها العدّاد المتصاعد، البطاقات الزجاجية، انتقالات الـ panel.

## 4) قواعد التطبيق (Do / Don't)
- ✅ **الأرقام أبطال:** حجم كبير، تباين عالٍ، Almarai ثقيل، عدّاد تصاعدي.
- ✅ **حركة هادفة فقط:** نبض النقطة الحمراء، سحب آلة الزمن، تصاعد العدّاد، فتح البانل — لا حركة زخرفية تشتّت.
- ✅ **RTL أصيل:** تخطيط معكوس صحيح (شريط يمين)، أرقام لاتينية مقبولة داخل العربية.
- ✅ **demo data موسومة دائماً** حتى وصول GeoJSON الحقيقي (الدستور §١٠·٥) — لا mock كحقيقي.
- ⛔ لا ألوان خارج لوحة الهوية · لا خرائط فاتحة · لا مكتبات خرائط مدفوعة · لا إضافة ميزات بعد H34.

## 5) إعداد سريع (Tailwind + RTL + خطوط)
- `tailwind.config` بـ tokens الهوية (الألوان + الخطوط + ظلال البطاقات الزجاجية).
- `dir="rtl"` افتراضياً + `font-family` عبر `@font-face` لـ Almarai/Tajawal (أو Google Fonts).
- مكوّنات الرسوم: **Recharts** لمنحنيات GRACE/NDVI/التنبّؤ.
- الحركة: مكتبة خفيفة (CSS/Framer Motion) — احذروا الوزن على لحظة العرض (إنترنت القاعة).

> **الأولوية القصوى للديمو:** **سير المفتّش يعمل بسلاسة من أول ضغطة لآخر ضغطة** (UI-10) — الصقل البصري بعده، لا قبله. *working > perfect*.
