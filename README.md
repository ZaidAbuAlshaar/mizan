## بروتوكول بدء الجلسات

الوثائق في الـ repo + ملف `CLAUDE.md` يلخص القواعد ويشير إليها — Claude Code يقرأها عند الحاجة بدل حرق ~25 ألف token كل جلسة. كل حساب يلصق **أمر الانطلاق الخاص به فقط** (`plan.md` §15). **القناة الرابعة للطوارئ:** Anthropic API key مدفوع عند احتراق أي حساب Max (TM-19).

## القواعد الذهبية

1. **working > perfect** — خط القطع: Phase 1 منتهية = نجاح مضمون.
2. **commit كل 30–60 دقيقة** — التأمين ضد rate limits.
3. **لا أحد يخترع رقماً** — كل رقم يتتبّع لجدول الأرقام (`plan.md` الملحق أ = الدستور §4).
