| المجلد | الدور |
|---|---|
| [`geo/`](geo/) | pipeline P1–P7 على Google Earth Engine + مولّد بيانات تجريبية (بلا GEE) |
| [`backend/`](backend/) | FastAPI — 11 endpoint (§12)، يقرأ `data/` الحقيقية أو `data/demo/` |
| [`frontend/`](frontend/) | Next.js 14 + Tailwind + MapLibre + Recharts — RTL، الشاشات الخمس |

تشغيل بأمر واحد: `bash scripts/dev.sh` · التفاصيل والبدائل (أوفلاين/GEE حقيقي): [`QUICKSTART.md`](QUICKSTART.md).

## بروتوكول بدء الجلسات

كل جلسة Claude Code تبدأ بلصق: **الدستور كامل + خطة الحرب + أمر الانطلاق الخاص بالحساب** (`plan.md` §15). لا شيء آخر.

## القواعد الذهبية

1. **working > perfect** — خط القطع: Phase 1 منتهية = نجاح مضمون.
2. **commit كل 30–60 دقيقة** — التأمين ضد rate limits.
3. **لا أحد يخترع رقماً** — كل رقم يتتبّع لجدول الأرقام (`plan.md` الملحق أ = الدستور §4).
